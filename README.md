# Girish Babu — Wellness & Transformation Portfolio

A premium, high-performance portfolio website for Girish Babu, a Wellness Coach specializing in lifestyle transformations. Built with a focus on rich aesthetics, smooth animations, and a seamless user experience.

![Girish Babu Portfolio](https://images.unsplash.com/photo-1571019613454-1cb2f99b2d8b?auto=format&fit=crop&q=80&w=1200)

## 🚀 Features

- **Premium Design System**: Tailored HSL color palette and sleek dark mode.
- **Fluid Parallax Effects**: Dynamic "My Story" section using GSAP and ScrollTrigger.
- **Interactive Gallery**: Smooth image lightbox and hover states.
- **Responsive Layout**: Optimized for all devices, from mobile to desktop.
- **Micro-animations**: Powered by Framer Motion for a lively interface.
- **Modern Tech Stack**: React, Vite, TypeScript, Tailwind CSS, and Shadcn UI.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏁 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Neurosensen/Girish-Babu.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Deployment

The project is ready for deployment on platforms like Vercel, Netlify, or GitHub Pages.

To build for production:
```bash
npm run build
```
